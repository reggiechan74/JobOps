02 Phase I: Arsenal Preparation

Acquire Target Coordinates

Upload job descriptions to prepare for reconnaissance and analysis.

Manual Upload or /searchjobs

What It Does

Before engaging any target, you need intelligence. This step involves acquiring job postings — either manually uploading them to the Job_Postings/ directory or using the automated job search function to discover opportunities.

Two Methods

Manual Upload

Copy job descriptions you find on LinkedIn, company websites, or job boards directly into markdown files.

Job_Postings/TechCorp_SeniorDev_2025.md

Automated Search

Use the job search command to discover and save relevant opportunities automatically.

/searchjobs "senior developer" Toronto --save

Search Command Options

  • +
    Keyword Search

    Search by job title, skills, or industry keywords.

  • +
    Location Filter

    Narrow results to specific cities, regions, or remote positions.

  • +
    Company Target

    Filter by specific company with --company=name flag.

  • +
    Auto-Save

    Use --save to automatically store results with full job descriptions.

File Naming Convention

Job_Postings/
CompanyName_Role_Date.md
SearchResults_Query_Date.md — Auto-saved search results

Usage Example

$ /searchjobs "product manager" "San Francisco" --limit=20 --save
Searching hiring.cafe for matches...
Found 20 positions, scraping full descriptions...
Saved: Job_Postings/SearchResults_ProductManager_2025-12-04.md
