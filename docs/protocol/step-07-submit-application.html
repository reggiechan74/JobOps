Convert to professional format and deploy. Multiple output options available.

/convert or /formatresume

What It Does

Converts your markdown resume and cover letter into submission-ready formats. Choose between Word DOCX for ATS compatibility or PDF for visual presentation.

Two Conversion Options

/convert — Word DOCX

Uses pandoc for professional Word document conversion. Best for ATS systems that parse DOCX files.

ATS-friendly Editable output Pattern matching

/formatresume — PDF

Uses Playwright for pixel-perfect PDF generation with professional themes. Best for visual presentation.

3 theme options Page control Visual polish

PDF Themes

Modern

Blue accents, contemporary sans-serif, clean lines

Classic

Serif fonts, formal structure, traditional elegance

Minimal

Maximum whitespace, focused content, clean aesthetic

Page Length Options

  • 1
    1 Page

    Entry-level or early-career. Concise and focused.

  • 2
    2 Pages

    Mid-career professionals. Standard length for most roles.

  • 3
    3 Pages

    Executive or senior technical. Comprehensive experience.

  • A
    Auto

    Content-based determination. System optimizes for fit.

Convert Command Patterns

resume — Convert latest Step 3 resume
coverletter — Convert latest cover letter
all — Convert resume + cover letter together
path/to/file.md — Convert specific file

Submission Strategy

After conversion, you're ready to submit. Remember:

  • Use DOCX for online application forms with parsing
  • Use PDF for direct email or uploads where formatting matters
  • Keep markdown source for future iterations
  • Track submission date for follow-up timing
